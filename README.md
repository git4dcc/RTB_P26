# RTB_P26
[![Real-time Bus (RTB)](https://img.shields.io/badge/RTB_Project-FF6699)](https://www.rtb4dcc.de)
[![Apache License 2.0](https://img.shields.io/badge/license-Apache%20License%202.0-blue)](https://www.apache.org/licenses/LICENSE-2.0)

The P26 composes a bread board for DCC/Railcom mobile decoder development. It contains nearly everything a mobile decoder needs - except the CPU.

> <img src="https://rtb4dcc.de/wp-content/uploads/2024/01/D16_1.png" width=400>
<br>

The decoder has the following features,
- DCC to 3.3V power circuit
- Inrush limiter
- Railcom sender circuit
- Motor driver H-bridge
- FL/FR LED lights

[more](https://rtb4dcc.de/hardware/modules/p26/)
